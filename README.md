# 🎓 AI Study Web – Website Học Tập Tích Hợp AI  
**Tác giả:** Nguyễn Văn Bảo Khánh  
**Công nghệ:** HTML • CSS • JavaScript • LocalStorage  

Website hỗ trợ học sinh lưu trữ tài liệu, tìm kiếm bài học, và sử dụng chatbot AI đơn giản để hỗ trợ học tập.  
Giao diện trực quan – dễ dùng – chạy hoàn toàn trên trình duyệt.

---

## 🌟 Tính năng chính

### 🏠 1. Trang chủ (Home)
- Giới thiệu trang web
- Điều hướng nhanh đến Thư viện / Chatbot / Liên hệ  
- Giao diện thân thiện, dễ dùng cho học sinh

### 📚 2. Thư viện tài liệu (Library)
- Upload tài liệu (PDF / hình ảnh / text tùy cấu hình)
- Lưu trữ bằng **LocalStorage** (không cần backend)
- Tìm kiếm tài liệu theo tên
- Hiển thị danh sách tài liệu theo dạng lưới

### 🤖 3. Chatbot AI đơn giản
- Chatbox nhỏ hỗ trợ học tập
- Gợi ý nội dung, giải thích lý thuyết cơ bản
- Mô phỏng AI bằng JavaScript (không gọi API)

### ✉️ 4. Liên hệ (Contact)
- Form gửi thông tin: tên – email – nội dung
- Xử lý bằng JS, có validate cơ bản

---
